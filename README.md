# 🎊 เว็บแฟนวันปีใหม่ (Vite + Tailwind CSS)

เว็บแฟนวันปีใหม่ที่สวยงาม สร้างด้วย Vite และ Tailwind CSS

## 📋 วิธีติดตั้ง

1. ติดตั้ง dependencies:
```bash
npm install
```

## 🚀 วิธีใช้งาน

### Development Mode (แนะนำ)
```bash
npm run dev
```
เซิร์ฟเวอร์จะเริ่มทำงานที่ `http://localhost:3000` และเปิดเบราว์เซอร์อัตโนมัติ

### Build สำหรับ Production
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## ✨ ฟีเจอร์

- ✨ หน้าเว็บสวยงามสำหรับวันปีใหม่
- 🎆 เอฟเฟกต์พลุและดาวตก
- 🎊 เอฟเฟกต์ confetti
- ⏰ นาฬิกานับถอยหลัง
- 📱 รองรับการแสดงผลบนมือถือ (Responsive)
- ⚡ Hot Module Replacement (HMR) ด้วย Vite
- 🎨 สไตล์ด้วย Tailwind CSS

## 🎨 เทคโนโลยีที่ใช้

- ⚡ **Vite** - Build tool ที่เร็วมาก
- 🎨 **Tailwind CSS** - Utility-first CSS framework
- 📦 **PostCSS** - CSS processing
- 🚀 **ES Modules** - Modern JavaScript

## 📁 โครงสร้างโปรเจกต์

```
├── index.html          # หน้า HTML หลัก
├── src/
│   ├── main.js        # JavaScript หลัก
│   └── style.css      # Tailwind CSS
├── vite.config.js     # Vite configuration
├── tailwind.config.js # Tailwind configuration
└── postcss.config.js  # PostCSS configuration
```
